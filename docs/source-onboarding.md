# Source Onboarding

The wizard at `/dashboard/sources/new` supports:

1. paste link or choose platform manually
2. auto-detect source type
3. confirm source type
4. choose sync mode
5. show credentials/setup requirements
6. save source
7. test connection
8. run initial sync or continue in demo mode
9. show success and sync run id

Reality check copy must remain visible: links identify the source; private metrics usually need credentials, OAuth, webhook secret, service role key, or tracking snippet.
