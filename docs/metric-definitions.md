# Metric Definitions

Metrics are defined in `src/aggregation/metric-definitions/definitions.ts`.

Primary MVP KPIs:
- `page_views`
- `unique_visitors`
- `custom_events`
- `signups`
- `users_total`
- `active_sources`
- `last_sync_status`
- `sync_errors`

Commerce metrics are scaffolded but not dominant until Shopify is connected.
